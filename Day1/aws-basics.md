# Day 1 – Introduction to Cloud & AWS Basics

## AWS Pricing Models

AWS provides different pricing models:

Pay-as-you-go  
You only pay for resources you actually use.

Reserved Instances  
You reserve capacity for 1–3 years and get discounts.

Spot Instances  
Unused AWS capacity offered at very low cost.

For startups or learners, Pay-as-you-go is best because it allows flexible scaling without large upfront investment.

---

## Cloud Deployment Models

On-Premises  
Infrastructure hosted inside a company’s own data center.

Cloud  
Infrastructure hosted by cloud providers like AWS.

Hybrid  
Combination of on-premises and cloud infrastructure.

Use cases:

Startups → Cloud  
Large enterprises → Hybrid  
Government systems → Often On-Prem or Hybrid
